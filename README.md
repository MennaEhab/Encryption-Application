# Encryption Application
 RC4 algorithm
 
 Description : 
               Encryption of a text up to 1000 ASCII characters, using a key consisting of 8 ASCII characters using the CR4 algorithm 
 
 Programming techniques:  
 - Using of MFC. 
 - Using classes (OOP) . 
Inputs :
  -ASCII key . 
  -ASCII text. 
Output :
  -encrypted bytes in hex format.
